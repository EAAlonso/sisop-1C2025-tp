# Trabajo Práctico: Parte 1 - Coordinación de Tareas con Procesos y Memoria Compartida
# GRUPO 12

## Descripción

Este proyecto simula el funcionamiento de una cocina digital usando un modelo cliente-servidor implementado en C++. Cada cliente representa un pedido de un combo, y el servidor actúa como una cocina con cocineros que procesan esos pedidos en paralelo.

La comunicación se realiza mediante sockets TCP y los mensajes se envían en formato JSON.
## 

## Ejecución

1. Servidor
./bin/app

2. Cliente
./bin/app

##

## Importante antes de ejecutar:

1. Ejecutar el comando "make" en la carpeta de servidor y cliente
2. Tal vez sea necesario instalar g++ (sudo apt install g++)
3. Si no le deja instalar g++, correr el comando: sudo apt update --allow-releaseinfo-change
